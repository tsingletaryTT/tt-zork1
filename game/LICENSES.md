## Adventure / Colossal Cave  (`advent.z5`)

Original game by Will Crowther (1975) and Don Woods (1977).

This Z-machine port is Graham Nelson's Inform 6 adaptation, Release 9 /
Serial 060321, based on Dave Baggett's TADS reconstruction "Colossal Cave
Revisited."  It is distributed from the Interactive Fiction Archive
(ifarchive.org) under terms compatible with the original authors' intent.

The canonical open-source release of the underlying Adventure source is
**Open Adventure** by Eric S. Raymond, released under the **BSD 2-Clause
License** with the explicit blessing of Will Crowther and Don Woods:

- https://gitlab.com/esr/open-adventure

The BSD 2-Clause License is compatible with the Apache 2.0 License under
which this repository is licensed.

## Note on `zork1.z3`

The `zork1.z3` file predates the November 2025 MIT license grant; its MD5
hash is identical to the compiled file in the `historicalsource/zork1`
repository (Release 119, Serial 880429), confirming it is the same binary
now covered by the MIT License.
