## Core Adventure Game Wisdom

**Rule 1: EXAMINE EVERYTHING**
- New room? → LOOK, then EXAMINE each object mentioned
- Found item? → TAKE it, then EXAMINE it
- Stuck? → EXAMINE what you haven't examined yet

**Rule 2: LIGHT IS LIFE**
- Never enter dark areas without a light source
- Conserve lamp battery (turn off in lit areas)
- Grues lurk in absolute darkness

**Rule 3: MAP SYSTEMATICALLY**
- Try all cardinal directions from each room
- Note which directions connect where
- "All alike" = you're in a maze (mark your path!)

**Rule 4: INVENTORY MANAGEMENT**
- Keep useful items (lamp, sword, keys)
- Drop heavy items you don't need immediately
- Remember where you dropped things

**Rule 5: READ EVERYTHING**
- Books, signs, inscriptions often contain hints
- The leaflet has important survival tips
- Some puzzles require reading clues
